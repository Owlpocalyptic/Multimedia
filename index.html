UI V A FP

Hello, and welcome to my portfolio! I'm Nicholas Mulrooney, aspiring software engineer and developer, and this is where I host my exploits in multimedia. These four projects are the fruits of many hours of toil in a Multimedia course hosted by Otago Polytechnic's Bachelor of Information Technology.

User Interface

A project to design the user interface for a relatively simple application. Done entirely in Adobe Illustrator, although this unit also included basic training in Photoshop. This project tested my ability to make informed decisions about an application's layout and appearance, so as to create the ideal user experience.

Video

For this project, we shot and edited a short film in groups of 1-3. I collaborated with Michael Lagocki and Joshua Sim on planning and shooting this film, before we splintered off to edit it individually. I used Adobe Premiere for my edit. The goal of this project was to demonstrate a clear understanding of all three aspects of the film-making process: pre-production, production, and post-production.

Animation

A 30-second long animation telling a short story. I drew and animated this entirely in Adobe Animate
